The Coord Grabber SaaS application is used to quickly identify the physical location in Military Grid Reference System (MGRS) of a point on the earth. This allows personnel that install hardware devices such as target, battlefield effects, wireless mesh devices, etc. on military training ranges to pinpoint the exact location of where to place hardware devices. The technology behind this application is an Angular application that integrates NASA's Web World Wind that allows the display of the earth and picking locations on the earth.

The application is hosted on an AWS deployment using the Elastic Container Service (ECS) utilizing Docker containers with Fargate. There is a CI/CD Pipeline setup with a GitHub Webhook that allows for continual integration and deployment when the application changes. Additionally, the docker images are stored on AWS's Container Registry.

This deployment is setup to scale horizontally out when there are lots of requests then scale back in when the requests have backed off. CoordGrabber uses CloudFront to host its deployment on the CDN for faster access. Additionally, it uses Route 53 with a custom domain secured by SSL/TLS certificates obtained by the Certificate Manager of AWS.
