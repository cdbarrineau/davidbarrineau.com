This website is a static website that is hosted on AWS. This site uses an S3 bucket to store the files that are served by AWS. In order to speed delivery of this site to end-users, the use of a CloudFront Distribution is used to provide a CDN close to where users access AWS resources. In keeping with best AWS practices, this site uses AWS's Origin Access Control to limit permissions when accessing the contents of the S3 bucket that contains the data for this site. Additionally, this site used a SSL/TLS certificate issued by AWS's Certificate Manager. Finally, the use of AWS's Route 53 is used to direct requests to the CloudFront distribution.
